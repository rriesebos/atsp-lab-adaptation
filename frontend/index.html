Welcome

This site serves as an example of current web vulnerabilities and their effects. Students and enthusiasts alike can use this setup in order to learn about these vulnerabilities, how they work and what can be done to prevent them. A github repository with the code for this project can be found by clicking the button below.

Github Repository

Welcome to Insecure Banking

Throughout the examples of this website we use the setting of an imaginary banking service called insecurebanking.com. This service provides a set of basic (insecure) operations that you can deal with and try to hack. The service consists of logging in and logging out capabilities, viewing your information and transactions, as well creating new transactions. These functionalities are covered in due time and spread out over the several pages. For each of the different web vulnerabilities covered there is a menu button on the top left.

NB: some of the operations in the examples are stateful and change the information stored in the database. At any point in time the database can be reset to its initial state with the reset button in the top-right corner.
