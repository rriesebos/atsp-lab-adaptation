How do we steal your money?

In order to steal the money from your bank account, we need to send a request to the actual banking application. Whenever a HTML page is loaded, the html is parsed and rendered on the screen. A html tag which is not that secure is the IMG tag. Normally, the IMG tag is used to tell the browser to render an image from a specific source. In order to render the image, the browser retrieves the image by sending a request to the source. However, this image source does not have to be a source of an image, but can also be an url of something else. The browser does not know the difference. This evil website contains one of these image tags. The source of the image contains is the following URL: 

                http://insecurebanking.com/api/transfer_money?to=5&amount=100&reference=CSRF Attack from evil website!

Whenever this page is loaded, this request is created. If the user is authenticated, money is actually transferred to the account of the attacker (account ID 5 in this case).
